# 🌴 Dashboard Visualisasi Data Perkebunan Indonesia

Dashboard interaktif untuk UAS Pengenalan Sains Data menggunakan **Streamlit**.

## 📋 Deskripsi
Dashboard ini menganalisis data produksi perkebunan 38 provinsi Indonesia 
untuk 7 komoditas (Kelapa Sawit, Kelapa, Karet, Kopi, Kakao, Teh, Tebu).
